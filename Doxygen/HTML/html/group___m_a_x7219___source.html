SPI Send function.

Note
In many commands, txMSB is the command address and txLSB is the command value. Last 16Bits of date are latched on rising edge of load pin. So it is not possible to send data without toggling load all 16 Bits.
Parameters
max7219pointer to the users max7219 data structure
txMSBMSB to be send (or command address)
txLSBLSB to be send (or command data)
Returns
none

1. Activate CS-Pin

2. Transmit Data

2. Deactivate CS-Pin
