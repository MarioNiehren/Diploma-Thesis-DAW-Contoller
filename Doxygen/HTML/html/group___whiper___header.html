void Whiper_update ( Whiper_structTd * Whiper)

Parameters
Whiperpointer to the users whiper structure
Returns
none

1. Check if a new ADC Value is available. if not -> leave function.

2. Reset Interrupt flag

3. Copy the new value to the sample array

4. Calculate the new smoothed value

5. Restart ADC conversion
