Detailed Description

This module contains helper functions for E-Paper specific SPI transmission with DMA.

DC and CS Pins will be handled by these helpers, so there is no need to handle them manually. These functions are optimized for the use with SPI E-Paper Modules.

Init Pin CS (Chip Select)

The pin will be initialized in inactive state as start condition.
Parameters

Portpointer to the GPIO Port of the CS Pin

  • GPIO_PIN_SET if the target device is high active
  • GPIO_PIN_RESET if the target device low active
Returns
none

1. Save CS Port to SPI structure

2. Save CS Pin to SPI structure

3. Save the state that is required to be active.

4. Check ActiveState and set Inactive State of the structure to the opposite.

5. Set Pin to inactive state
