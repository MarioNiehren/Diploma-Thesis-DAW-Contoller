◆ PID_update()

void PID_update ( PID_structTd * PID,
double sample )

Call this function periodically calculate the newest PID output. Be careful to call this function faster as the set sample time, otherwise it might get difficult to calibrate the PID-terms.

Parameters
PIDpointer to the users PID structure
Sampleis the current process variable
Returns
result of the PID calculation with the new sample.

1. Check it Sample-Time elapsed. Leave function if not!

2. Calculate error between set point and current sample

3. Calculate proportional Term

4. Calculate integral Term with anti wind up

5. Calculate derivative Term with low pass to avoid noise

6. calculate output value

7. limit output value

8. save output to users PID structure (raw and round)
