◆ Buffer_get_TxSize()

uint16_t Buffer_get_TxSize ( MIDI_buffer_structTd * Buffer)

Get the length of the Array to be send.

Warning
This function must be called after Buffer_get_TxStartPtrForTransmission(), otherwise it will return the size of the wrong buffer. This function can only be called once for each cycle because the index counter will be reset.
Parameters
Bufferpointer to the users Buffer
Returns
size of the buffer that will be send. 0 if the buffer is empty or in case of an error.

◆ Buffer_get_TxStartPtrForTransmission()

uint8_t * Buffer_get_TxStartPtrForTransmission ( MIDI_buffer_structTd * Buffer)

Get the Buffer Array, that is ready to send. The returned array will be locked until this function gets called again.

Warning
Call this function exactly once for each transmission cycle. Otherwise the wrong array will be locked.
Parameters
Bufferpointer to the users Buffer
Returns
Start pointer of the buffer to be send, or NULL in case of an error
