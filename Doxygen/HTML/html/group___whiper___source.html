◆ get_SmoothedVlaueWithHysteresis()

uint16_t get_SmoothedVlaueWithHysteresis ( Whiper_structTd * Whiper,
uint16_t SamplesAverage )

Check if the average of samples is in a hysteresis area around the current value. It reduce noise a lot but the hysteresis threshold should be as low as possible to avoid big value jumps around the current value.

Parameters
Whiperpointer to the users whiper structure
SamplesAverageaverage value calculated from existing samples
Returns
valid smoothed value. If the hysteresis is inactive, it will be equal to Sampled Average.

◆ store_SampleFromDMABuffer()

void store_SampleFromDMABuffer ( Whiper_structTd * Whiper)

Copy the current value from the DMA buffer to the Sample Array.

Parameters
Whiperpointer to the users whiper structure
Returns
none

1. Copy Value from DMA buffer to Sample Array at current Array-Index

2. Count up the sample array index for the next value.

3. If sample array index reaches the defined NUMBER_OF_SAMPLES, reset the index. The following values will overwrite the old ones.

void Whiper_start ( Whiper_structTd * Whiper)

Function to start the ADC over DMA conversion. Call this function once when the ADC should start.

Parameters
Whiperpointer to the users whiper structure
Returns
none

1. Declare variable with the length of bytes to be buffered by DMA. We have 12Bit ADC, so we need 2 bytes.

2. Start ADC in DMA mode and link whiper->value to DMA to buffer data from Peripheral to Memory

◆ Whiper_update()

void Whiper_update ( Whiper_structTd * Whiper)

Call this function periodically in while function to check the whiper if new ADC values are available and to smooth them.

Parameters
Whiperpointer to the users whiper structure
Returns
none

1. Check if a new ADC Value is available. if not -> leave function.

2. Reset Interrupt flag

3. Copy the new value to the sample array

4. Calculate the new smoothed value

5. Restart ADC conversion
