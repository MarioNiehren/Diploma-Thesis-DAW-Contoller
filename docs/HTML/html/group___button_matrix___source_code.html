◆ ButtonMatrix_update()

void ButtonMatrix_update ( buttonMatrix_structTd * buttonMatrix)

Main function to update the button matrix.

Parameters
buttonMatrixPointer to the structure of the button matrix setup by the user
Returns
none TODO: Add Ret true if matrix is active. This can be used to mark if the user should fetch new states.

1. check if the debounce timer is elapsed.

  • if not: leave function

2. check if the matrix is in read mode

  • if not: check for interrupt and enter read mode if an interrupt is detected.

3. check again if the matrix is in read mode

  • if not: leave function

4. Read the button states on the active drive line (according to the buttonMatrix_Td structure)

5. check if all drive lines were processed

  • if not: count active drive line up and leave function.

6. Manage interrupt lines: get the active interrupt lines and deactivate every line, that is not active anymore.

7. check if any button still is pushed

  • if not: reset drive pins and leave read mode

8. Count up the active drive line in buttonMatrix_structTd structure and leave function

◆ check_InterruptDetected()

bool check_InterruptDetected ( buttonMatrix_structTd * buttonMatrix)

Checks if any interrupt line is interrupted.

Parameters
buttonMatrixPointer to the structure of the button matrix setup by the user
Returns
true if an interrupt was found

1. Set the return value with default false.

2. Use bitwise AND on the register that flags the interrupted GPIOs with the register that flags the digits in account of the other register. If a relevant flag is set (!= 0), set the return value to true.

3. Return the return value.

◆ get_InterruptLinesStateRegister()

uint16_t get_InterruptLinesStateRegister ( buttonMatrix_structTd * buttonMatrix)

Get the Register that describes the active interrupt lines.

Parameters
buttonMatrixPointer to the structure of the button matrix setup by the user
Returns
uint16_t value that shout be interpreted as a bit register.

1. setup empty register that stores active interrupt lines to be written and returned

2. loop through all interrupt lines

3. in loop: check if any button flag is set on the current interrupt line.

  • if yes: Set the active flag for the interrupt line using the interrupt pin.

4. Return the register with all active interrupt line flags. If no Line is active, the value will be 0

◆ read_ButtonsOnActiveDriveLine()

void read_ButtonsOnActiveDriveLine ( buttonMatrix_structTd * buttonMatrix)

Read all button states on an active drive line and flag them in the corresponding interrupt line register.

Parameters
buttonMatrixPointer to the structure of the button matrix setup by the user
Returns
none

1. Set the currently active drive pin to low

2. Loop through all existing interrupt lines

3. In the loop: Check if the currently relevant interrupt pin was interrupted. If yes:

  • Read the currently relevant interrupt pin
  • set or reset the Pin Flag in the interrupt line register with bit manipulation according to the previously read state

4. Set the currently active drive pin back to high
