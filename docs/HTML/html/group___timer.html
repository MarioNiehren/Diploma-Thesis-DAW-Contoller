User Functions

Functions to use the timer in any user code

How to use the timer:

  1. Declare an object of timer_setup_structTd type
  2. Set the threshold time (The rest of the structure will be set to 0 here)
  3. Check it the timer is elapsed. The Timer will automatically restart if the function returns true.
Note
This timer is based on the HAL_GetTick() function. It this function is not setup to milliseconds, the threshold will have an other unit. HAL-Standard is in milliseconds. Only modify this if a smaller interval than milliseconds is needed as it my require a deeper dive to the HAL-Drivers.
