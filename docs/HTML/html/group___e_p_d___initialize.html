Detailed Description

These functions are used to initialize the E-Paper Display.
back to EPD main page.

How to use:

  1. Setup an Object of Type EPD_StructTd
  2. Initialize Pins for DC, CS, RST, and BUSY
  3. Initialize SPI handler
  4. Initialize EPD

void EPD_init ( EPD_StructTd * EPD)

Initialize E-Paper. This will link parts of the EPD structure to some helper structures and sets EPD-structure to proper start values.

Parameters
EPDpointer to the users EPD structure
Returns
None

1. Init width and heigth of EPD for GUI Module.

2. Link Frame Buffer to GUI Module.

3. link Command Buffer to the Buffer helper.

4. link Data Buffer to the Buffer helper.

5. link Frame Buffer to the Buffer helper.

6. link LUT Buffer to the Buffer helper.

7. link Descriptor array to the Buffer helper.

8. link GUI structure of EPD structure to GUI Module

9. Set start conditions of EPD Structure

  • high to send data
  • low to send commands
    Parameters
    EPDpointer to the users EPD structure
    GPIO_DCGPIO Port of the DC Pin
    PIN_DCPin of the DC Pin
    Returns
    None

1. Set DC Pin for the SPI communication. SPIDMA_init_PinDC() is used for this. Argument 3 of the function is GPIO_PIN_RESET as this is required by the EPD-driver to be in Command Mode (which is default in our case).

◆ EPD_init_SPI()

void EPD_init_SPI ( EPD_StructTd * EPD,
SPI_HandleTypeDef * hspi )

Link STM32 DMA-SPI HAL-handle. The SPI handle is generated by Cube MX as a part of the STM32-HAL.

HAL Settings for SPI (as tested):

  • SPI Mode: Transmit Only Master
  • Hardware NSS Signal: Disable
  • Parameter Settings
    • Basic Parameters
      • Motorola
      • 8 Bits
      • MSB First
    • Clock Parameters
      • CPOL: High
      • CPHA: 2 Edge
  • NVIC Settings:
    • Enable Interrupt for DMA and SPI
  • DMA Settings
    • DMA Request: SPI1_TX
    • Channel: DMA1 Channel 3
    • Direction: Memory To Peripheral
    • Priority: Low
Parameters
EPDpointer to the users EPD structure
hspipointer to the HAL-SPI-Struct
Returns
None

1. Link hspi to the SPI-helper.
