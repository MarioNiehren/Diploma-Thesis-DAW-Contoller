Detailed Description

This module is used to calculate a PID output to reach a specific and stable target.

How to use:

  1. Declare an object of PID_structTd data type
  2. Initialize this object with init functions
  3. Use set functions to tune PID.
    • Set Kp, Ki, Kd
    • Set Low Pass
    • Set Limits for output value
    • Set Sample Time for controlled update frequency
  4. Set Target
  5. In while loop:
    • Get the current value of the controlled system
    • Update PID with the current value
    • Use Get-Functions to get the PID results and use them to correct the system that is controlled.

A manual approach for calibration:

  1. Set Kp until the output gets as close as possible to the target without overshooting
  2. Set Ki to come closer to the target. Try to find a good balance between speed and overshooting
  3. Set Kp if it is required to reach the target faster. Be careful because the D term tends to bring noise and heavy overshooting to the system. Many systems do not require a D-Term.
