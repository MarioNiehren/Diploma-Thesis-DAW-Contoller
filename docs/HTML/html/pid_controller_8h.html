Use these functions to set or reset parameters

void PID_set_OutputMinMax (PID_structTd *PID, double Min, double Max)
 Set the value range for the PID output.
 
void PID_set_KpKiKd (PID_structTd *PID, double Kp, double Ki, double Kd)
 Set the coefficients for proportional, integral and derivative terms.
 
void PID_set_Target (PID_structTd *PID, double setpoint)
 Set new set point for the PID Controller as new target of the controlled system.
 
void PID_set_LowPass (PID_structTd *PID, double Tau)
 Set tau to tune the low pass used on the D-Term to reduce noise. Increase tau for a lower crossover frequency. Be careful that the D-Term does not get "killed" by this filter.
 
void PID_set_SampleTimeInMs (PID_structTd *PID, uint32_t Threshold)
 Set the sample time to control the update frequency of the PID-controller. Make sure that the program is fast enough to call the update function at least as fast as the sample time. Otherwise the sample time will be uncontrolled slower.
 
void PID_reset (PID_structTd *PID)
 Reset all values from previous calculations in the structure to make the PID-controller ready for a clean start.
 
Process

Use these functions to calculate the PID

void PID_update (PID_structTd *PID, double sample)
 Call this function periodically calculate the newest PID output. Be careful to call this function faster as the set sample time, otherwise it might get difficult to calibrate the PID-terms.
